Intro To HTML and CSS

HTML (Hypertext Markup Language) structures content on the web. It's used to define elements like headings, paragraphs, images, and links.

CSS (Cascading Style Sheets) adds styling, such as colors, fonts, and layout, to make the content visually appealing.

console.log("Hello, World!");
HTML Basics

HTML Structure: Begins with <!doctype html>, as the root, and includes <head> for metadata and <body> for visible content. Each tag needs to be closed with the content in the middle, example below:

<!doctype html>

<head>

metadata goes here..

</head> -> closing tag

<body>

visible content goes here..

</body> -> closing tag

</html> -> dont forget to close the very first html tag, this should come last
CSS Basics

CSS Syntax: Consists of selectors, properties, and values, e.g., p { color: red; }.

CSS Application

Inline: Directly in HTML elements (e.g., This text is red).

Internal: In the <style> tag within the <head>

External: In a separate .css file, linked with <link> in the <head>.

var name = "John";
Text Styling with CSS

Fonts: Control typeface (font-family), size (font-size), and weight (font-weight).

Color: Use color for text color, background-color for background.

Alignment: Use text-align to align text left, center, or right.

Spacing: line-height for vertical spacing, letter-spacing for space between letters.

Layouts and Positioning

Box Model: Elements are treated as boxes with content, padding, border, and margin.

Display:

  • block: Takes full width.
  • inline: Fits content width.
  • inline-block: Combines block and inline.

Positioning:

  • static: Default positioning.
  • relative: Moves based on its normal position.
  • absolute: Positioned relative to the nearest positioned ancestor.
  • fixed: Stays fixed on the screen.

Float and Clear: Floats elements (e.g., images) to the left or right; clear to prevent overlap.

for (let i = 0; i < 5; i++) { console.log(i); }
Responsive Design

Media Queries: Adjust styles based on screen size, e.g., @media (max-width: 600px).

Flexible Units: Use percentages, vw, and vh for responsive sizing.

Flexbox: A layout model with flexible, responsive containers:

  • display: flex; for a container.
  • Align items with justify-content and align-items.

CSS Grid: Another layout system for more complex designs with rows and columns.

Working with Colors and Backgrounds

Colors: Defined with Hex (#ff0000), RGB (rgb(255, 0, 0)), or HSL (hsl(0, 100%, 50%)).

Backgrounds:

  • Background-color: Sets background color.
  • Background-image: Adds an image as background.
  • Position and Repeat: Control image alignment and repetition.

Gradients: Linear (linear-gradient()) and radial (radial-gradient()) gradients add smooth color transitions.

CSS Transitions and Animations

Transitions: Smoothly change properties over time using transition: property duration timing-function;.

Animations:

  • Define with @keyframes for complex sequences.
  • Example: @keyframes slide { from { transform: translateX(0); } to { transform: translateX(100px); } }.
  • Apply with animation: slide 2s infinite;.
