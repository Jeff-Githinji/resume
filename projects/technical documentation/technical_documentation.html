Intro To HTML and CSS

HTML (Hypertext Markup Language) structures content on the web. It's used to define elements like headings, paragraphs, images, and links.

CSS (Cascading Style Sheets) adds styling, such as colors, fonts, and layout, to make the content visually appealing.

<h1>Welcome to my website</h1> <p>This is styled with CSS</p>
HTML Basics

HTML Structure: Begins with <!doctype html>, <html> as the root, and includes <head> for metadata and <body> for visible content.

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>My Page</title> </head> <body> <h1>Hello World</h1> </body> </html>
CSS Basics

CSS Syntax: Consists of selectors, properties, and values.

CSS Application Methods:

1. Inline: Directly in HTML elements

2. Internal: In the <style> tag within the <head>

3. External: In a separate .css file, linked with <link>

/* External CSS example */ h1 { color: blue; font-size: 24px; } .container { max-width: 1200px; margin: 0 auto; }
Text Styling with CSS

Common text styling properties in CSS:

.text-example { font-family: Arial, sans-serif; font-size: 16px; font-weight: bold; color: #333333; text-align: center; line-height: 1.6; letter-spacing: 1px; }
Layouts and Positioning

Box Model: Elements are treated as boxes with content, padding, border, and margin.

.box-model-example { width: 200px; padding: 20px; border: 2px solid #333; margin: 10px; } .positioning-example { position: relative; top: 10px; left: 20px; } .float-example { float: left; width: 50%; } // ... rest of the content remains the same ...
Responsive Design

Media Queries and Flexible Layouts:

/* Media Query Example */ @media (max-width: 768px) { .container { width: 100%; padding: 0 15px; } } /* Flexbox Example */ .flex-container { display: flex; justify-content: space-between; align-items: center; } /* Grid Example */ .grid-container { display: grid; grid-template-columns: repeat(3, 1fr); gap: 20px; }
Working with Colors and Backgrounds

Colors and Background Properties:

.color-examples { color: #ff0000; /* Hex */ color: rgb(255, 0, 0); /* RGB */ color: hsl(0, 100%, 50%); /* HSL */ } .background-examples { background-color: #f0f0f0; background-image: url('image.jpg'); background-position: center; background-repeat: no-repeat; background-size: cover; } .gradient-example { background: linear-gradient(to right, #ff0000, #00ff00); }
CSS Transitions and Animations

Examples of transitions and animations:

/* Transition */ .button { background: blue; transition: background 0.3s ease; } .button:hover { background: darkblue; } /* Animation */ @keyframes slide { from { transform: translateX(0); } to { transform: translateX(100px); } } .animated-element { animation: slide 2s infinite; }
Conclusion

Recap: HTML structures content, while CSS styles and positions it for visual appeal. Together, they form the foundation of web development.

Next Steps: Learn more about JavaScript for interactivity and explore frameworks like Bootstrap for advanced styling.
